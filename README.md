# renderorange-gnome-shell-extension

## installation

```
mkdir -p ~/.local/share/gnome-shell/extensions
mkdir ~/src
git -C ~/src clone https://github.com/renderorange/renderorange-gnome-shell-extension.git
ln -s ~/src/renderorange-gnome-shell-extension ~/.local/share/gnome-shell/extensions/renderorange@renderorange
```

reload gnome-shell
enable in gnome-extension-manager
